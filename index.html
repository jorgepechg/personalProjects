¿Qué es el desarrollo web?

El desarrollo web es en términos sencillos todo lo relacionando con el diseño, creación y mantenimiento de software e infraestructura para la web, esto incluye desde páginas web de texto simple hasta aplicaciones web complejas.

El objetivo del desarrollo web es lograr una buena experiencia de usuario cumpliendo con los requerimientos de cada proyecto. El buen desarrollo web procura que sus sitios sean estables y eficientes, garantizando seguridad, rapidez, disponibilidad y usabilidad.
